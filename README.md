# DriveBidRent

A full-stack application for bidding, renting, and managing vehicles.

## Features

- User authentication and authorization
- Auction management for buyers and sellers
- Rental services
- Admin dashboard
- Mechanic assignments
- Chat functionality

## Tech Stack

- **Backend**: Node.js, Express.js, MongoDB
- **Frontend**: React, Vite
- **Authentication**: JWT
- **File Upload**: Cloudinary

## Installation

### Backend

1. Navigate to the Backend directory:
   ```
   cd Backend
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Set up environment variables in a `.env` file.

4. Start the server:
   ```
   npm start
   ```

### Frontend

1. Navigate to the client directory:
   ```
   cd client
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm run dev
   ```

## Usage

- Register as a buyer, seller, or admin.
- Post auctions or rentals.
- Bid on items or rent vehicles.
- Manage your profile and earnings.
